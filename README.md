# Weather-App
A simple weather application that fetches and displays current weather and hourly forecasts based on the user's input. The app uses the OpenWeatherMap API to retrieve weather data and presents it in a user-friendly format.

## Features

- Search for weather information by city, province/state, and country.
- Display current temperature, weather description, and weather icon.
- Show an hourly forecast for the next 24 hours.
- Responsive design for an optimal user experience across devices.

## Installation

1. **Clone the repository:**
    ```bash
    git clone https://github.com/yourusername/weather-app.git
    
2. **Navigate to the project directory:**
    ```bash
    cd weather-app

3. **Open `index.html` in your browser.**  
   No additional setup is required as this project does not involve a backend server or additional dependencies.


## Usage
1. Enter the city, province/state, and country into the search fields.
2. Click the "Search" button to retrieve weather information.
3. View the current temperature, weather description, and hourly forecast.


## API Key
This application uses the OpenWeatherMap API. You need to replace the `apiKey` variable in `script.js` with your own API key.  
1. Sign up for an API key at [OpenWeatherMap](https://openweathermap.org/api).
2. Replace `apikey goes here` with your API key in `script.js`.

## Files
- `index.html` - The main HTML file for the app.
- `style.css` - The CSS stylesheet for styling the app.
- `script.js` - The JavaScript file containing the logic for fetching and displaying weather data.
